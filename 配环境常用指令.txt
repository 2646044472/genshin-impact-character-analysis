cuda:
conda install -c nvidia/label/cuda-11.8.0 cuda-toolkit=11.8  锁11.8所有的相关依赖
conda install -c nvidia cuda-toolkit=11.2  会装一些不相关的依赖，不推荐
conda install cuda=12.1.0 -c nvidia  会装一些不相关的依赖，不推荐
conda list | grep cuda
